Android Edge AI Runtime for Unity

A Unity runtime integration for executing compatible local GGUF language models on supported Android devices. It connects Unity C# code directly to a native C++ runtime without requiring a separate inference server.

01

Local inference

Run compatible local language models directly on supported Android hardware.

Typical workflow

From Unity project to Android device

  1. 1Import the package into a Unity project.
  2. 2Add and configure the runtime component.
  3. 3Set the model path and runtime parameters.
  4. 4Build for a supported Android arm64 device.
  5. 5Run inference locally from the Unity application.

Important product notes

GGUF model files are not included. Actual model compatibility, performance, and memory requirements depend on the selected model, Android device, available memory, and project configuration.
